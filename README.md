# Schedule

## Installation
- ##### Create venv

   ```python3 -m venv .\venv```
- ##### Activate venv
 
   ```.\venv\Scripts\activate```
- ##### Install requirements
 
   ```pip install requirements.txt```

## Usage
- ##### Run script

   ```python3 main.py```
   
- ##### Login in your google account

- ##### Enter your group id
